Speech denoising system using Recurrent Neural Networks (specifically LSTMs) to predict spectral mask parameters to apply to audio using DSP, to remove noise.

Author: Skylar Denno
Dataset from: University of Edinburgh, School of Informatics. Centre for Speech Technology Research (CSTR)
https://doi.org/10.7488/ds/2117

The src folder contains the source code for each of the scripts, sequentially numbered in the order they should be run for the full development pipeline.
For each source file, there exists a config file with the same name (.yaml) in the config folder.
